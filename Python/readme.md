## Objectif
Collection d'implémentations d'algorithmes classiques en Python,
accompagnée d'exemples et de tests. Idéal pour démontrer tes compétences en algorithmique,
structures de données, et préparation aux études avancées.

## Contenu
- `sorting.py` : tri rapide, tri fusion, tri par insertion (et comparaisons).
- `search.py` : recherche binaire.
- `graph_basic.py` : DFS, BFS, Union-Find.
- `graph_mst.py` : Kruskal, Prim.
- `graph_flow.py` : Ford-Fulkerson (Edmonds-Karp).
- `shortest_paths.py` : Dijkstra, Bellman-Ford.
- `dp.py` : 0/1 knapsack, Longest Increasing Subsequence (LIS).
- `strings.py` : KMP (Knuth-Morris-Pratt), Trie.
- `geometry.py` : Graham scan (convex hull).
- `data_structures.py` : Fenwick (BIT), Segment Tree.
- `tests/test_algorithms.py` : tests simples pour chaque algorithme.
