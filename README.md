This repository contains the implementation in C, Java and Python of the following algorithms

-- Tri

Quicksort (in-place, O(n log n) moyen)

Mergesort (O(n log n))

Insertion sort (O(n²))

-- Recherche

Recherche binaire

-- Graphes

BFS, DFS itératif

Union-Find (path compression + union by rank)

Kruskal (Arbre couvrant minimal)

Dijkstra

Bellman-Ford

-- Programmation dynamique

Sac à Dos 0/1

LIS (Longest Increasing Subsequence)

-- Structures de données

Fenwick Tree (BIT)

Trie

Segment naïf via adj list & min-heap dans Dijkstra

-- Géométrie algorithmique

Enveloppe convexe (Graham Scan)
